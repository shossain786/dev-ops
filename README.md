# Sample UI App

This repository contains a simple static UI application built with HTML, CSS, and JavaScript.

## Files

- `index.html` - Page structure and app layout
- `styles.css` - Responsive UI styling
- `app.js` - Task tracker logic and behavior

## Run locally

Open `index.html` in any browser, or use a local static server.

### Using Python

```bash
python3 -m http.server 8000
```

Then navigate to `http://localhost:8000`.
